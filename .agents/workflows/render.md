# Workflow: 3D Render Prompt (/render)

This workflow is triggered when the user executes the command `/render` or `/render [subject]`.

## Execution Steps:
1. **Simulation Technology**: Use software simulation keywords (Octane render, Unreal Engine 5, Raytracing, Path tracing).
2. **Studio Lighting**: Clearly define the light sources (Three-point lighting, rim light, softbox, HDRI environment).
3. **Material Details**: Describe hyper-realistic 3D materials (subsurface scattering, glossy reflections, matte finish).
4. **Composition**: Focus heavily on the central subject with a clean background (e.g., dark studio background, pure white infinite backdrop).
5. **Output**: Return an optimized prompt for Product Render or 3D Concept images.
