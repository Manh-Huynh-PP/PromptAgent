# Workflow: Timecode Storyboard (/storyboard)

This workflow is triggered when the user executes the command `/storyboard` or `/storyboard [idea]`.

## Execution Steps:
1. **Time Segmentation**: Break down the idea into clear timecodes using the `[00:00-00:xx]` format.
2. **Develop Action**: Write continuous actions for each timecode. Ensure logical spatial and temporal transitions.
3. **Audio Directing**: Add `SFX: [sound]` or `Ambient noise: [sound]` tags synchronized with the on-screen action.
4. **Identity Continuity**: Ensure characters and objects maintain the exact same descriptions (clothing, material, color) across all segments.
5. **Output**: Present the storyboard script in a clear, professional format.
